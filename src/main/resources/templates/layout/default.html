우리 자녀들에게 보여주고 싶은 세계 유수의 미술관, 어떻게 제대로 즐길 수 있을까?
이번 여름방학에는 자녀들과 함께 극장으로 미술관 나들이 오세요.
어린이의 눈높이에 맞춰 쉽고 재미있게 예술 이야기를 전달할 4인의 현직 도슨트들이 기다립니다.

    [로열 발레] 잠자는 숲속의 미녀

    마법과 동화 캐릭터가 펼쳐지는 가족 특선 발레, 잠자는 숲속의 미녀 차이콥스키의 아름다운 음악과 올리버 메셀의 화려한 무대가 황홀한 동화의 세계로 이끕니다. [상영 정보] ㆍ상영지점 : 서울 = 더 부티크 목동현대백화점 / 상암월드컵경기장 / 성수 / 센트럴 / 코엑스 경기/인천 = 부천스타필드시티 / 분당 / 안성스타필드 / 백석벨라시타 / 킨텍스 / 하남스타필드 / 남양주 / 영종 대전/충청 = 대전신세계 아트앤사이언스 전라 = 전주혁신 부산/대구/경상 = 대구이시아 / 마산 / 부산대 / 울산 / 양산증산 ㆍ상영일정 : 2023년 7월 3일(월) ~ 2023년 7월 23일(일) 매주 월요일 / 일요일 상영 ㆍ러닝타임 : 180분 (인터미션 없음) [제작진 정보] 안무 : 마리우스 페티파 (Marius Petipa) 음악 : 표트르 일리치 차이콥스키(Pyotr Il’yich Tchaikovsky) 오리지널 디자인 : 올리버 메셀(Oliver Messel) 지휘 : 코엔 케셀스(Koen Kessels) [출연진 정보] 오로라 공주 : 야스민 나그디(YASMINE NAGHDI) 플로리문드 왕자 : 매튜 볼(MATTHEW BALL) [작품 소개] 마법과 동화 캐릭터가 펼쳐지는 가족 특선 발레 잠자는 숲속의 미녀는 로열 발레단 역사에서 심장과도 같은 특별한 작품이다. 이 작품은 세계 2차 대전 후인 1946년 로열 오페라 하우스가 코벤트 가든에서 재오픈할 당시, 로열 발레단이 가장 처음 공연한 발레 작품이다. 2006년 잠자는 숲속의 미녀의 오리지널 무대가 다시 무대에 올랐고, 이후로 관객들의 사랑을 꾸준히 받아왔다. 프레데릭 애슈턴은 19세기 마리우스 프티파의 순수 클래시시즘을 끌어내 로열 발레단만의 잠자는 숲속의 미녀를 만들어 냈다. 차이콥스키의 아름다운 음악과 올리버 메셀의 화려한 무대 그리고 클래식 발레 레퍼토리의 정석인 발레 잠자는 숲속의 미녀는 관객을 황홀한 동화의 세계로 빠져들게 한다. [시놉시스] 왕 프로레스탄 24세와 그의 왕비는 그녀의 딸, 오로라 공주의 탄생을 축하하기 위해 모든 요정들을 초대한다. 공주 탄생 축하연은 마녀 카라보스의 등장으로 방해를 받는다. 초대받지 못해 화가 난 카라보스는 오로라 공주가 미래에 손가락을 물레 바늘에 찔려 죽게 될 것이라고 저주한다. 라일락 요정은 카라보스 저주에 대항해 오로라 공주가 죽지 않고 깊은 잠에 빠져 훗날 왕자의 키스에 의해 카라보스 저주에서 풀려날 수 있도록 축복의 마법을 건다. 오로라 공주의 16세 생일날 4명의 왕자들은 구혼한다. 이때, 한 노파가 오로라 공주에게 물레 바늘을 건네주고 오로라 공주는 손가락이 찔려 잠이 든다. 라일락 요정은 파티에 참석한 모두가 깊은 잠에 빠지도록 마법을 걸고, 궁전은 깊은 숲속으로 사라진다. 100년 후, 왕자 플로리문드는 시종들과 함께 숲에서 사냥을 한다. 라일락 요정이 나타나 그에게 오로라 공주의 모습을 보여준다. 플로리문드 왕자는 라일락 요정에게 자신을 오로라 공주에게 데려다 달라 애원한다. 마침내, 플로리문드 왕자는 오로라 공주를 키스로 깨우고 저주와 함께 카라보스도 사라진다.
